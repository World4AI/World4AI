Blocks

World4AI is organized into building blocks. Each block builds knowledge that is fundamental to modern artificial intelligence. But you don't need to be proficient in any of the topics to start the next one and you do not need to learn the blocks in a stricly linear manner. You can learn the material that you require as you go along.

At the moment only a few sections have been started and none is considered finalized. More chapters will be published in the next weeks and months. But the overall process will most likely take many years.
