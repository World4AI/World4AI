MDP as Sequential Interaction

In essence a Markov decision process (MDP) allows us to formalize the sequential interaction loop between the agent and the environment. The common assumption in reinforcement learning is the existence of a MDP at the core of each environment.

A Markov decision process (MDP) is a formal description of a sequential decision problem with uncertainty.

The interaction is done sequentially, where the agent and the environment take turns to react to each other. Each iteration of actions, rewards and states happens in a period of time, called a time step, t undefined. The time step is a discrete variable starting at 0 and increasing by 1 after each iteration. During the first time step the agent receives the initial state of the environment S_0 undefined and reacts accordingly with the action A_0 undefined . The environment transitions into a new state S_1 undefined and generates the reward R_1 undefined. The agent in turn reacts with the action A_1 undefined and the interaction continues. The general notation of writing States, Actions and Rewards is S_t, A_t, R_t undefined where the subscript t undefined represents a particular time step.
