Forward Pass

In the forward pass the features in the dataset are processed layer by layer, where the outputs from the previous layer are used as inputs into the current layer. As the name forward pass suggests, information flows unidirectionally from the input layer to the output layer.

You can use the interactive example of a neural network below to better understand the calculation of a neural network. This is the same architecture from the previous section that we are going to use to create a nonlinear decision boundary. The leftmost (red) numbers are the two features that are used as the input into the neural network. The network has two hidden layers with four and two neurons respectively. The neurons in the same layer use the same inputs, but have individual weights, which allows the neural network to learn different representations based on the same input. The single output represents the probability to belong to the category 1. Each neuron uses the sigmoid as its activation function. By selecting one of the neurons you can observe which weights, inputs and bias were used in the calculation. The table of the neural network is designed to show the exact values and intermediary steps that were used to calculate the output of a particular unit.

Variable Value
Input x_0 undefined 0.50
Weight w_0 undefined -0.32
Input x_1 undefined 0.50
Weight w_1 undefined 0.01
Bias b undefined 0.78
Net Input z = \sum_m w_j x_j + b undefined 0.63
Neuron Output a = \dfrac{1}{1 + e^{-z}} undefined 0.65

While the interactive example is a good way to demonstrate the functionality of the neural network, we need a way to formalize these calculations through mathematical notation.

In the above example we worked with a single sample, but in practice in the forward pass we will use a dataset consisting of many samples, a mini batch \mathbf{X} undefined. As usual \mathbf{X} undefined is an n \times m undefined matrix, where n undefined (rows) is the number of samples and m undefined (columns) is the number of input features.

\mathbf{X} = \begin{bmatrix} x_1^{(1)} & x_1^{(2)} & x_1^{(3)} & \cdots & x_1^{(m)} \\ x_2^{(1)} & x_2^{(2)} & x_2^{(3)} & \cdots & x_2^{(m)} \\ x_3^{(1)} & x_3^{(2)} & x_3^{(3)} & \cdots & x_3^{(m)} \\ \vdots & \vdots & \vdots & \cdots & \vdots \\ x_n^{(1)} & x_n^{(2)} & x_n^{(3)} & \cdots & x_n^{(m)} \\ \end{bmatrix} undefined

Each neuron in a particular layer has the same set of inputs, but has its own set of weights \mathbf{w} undefined and its own bias b undefined. For convenience it makes sence to collect the weights in a matrix \mathbf{W} undefined and the biases in the vector \mathbf{b} undefined. We distinguish between layers by using the superscript <l> undefined , where l undefined represents the number of the layer. The matrix \mathbf{W}^{<1>} undefined for example contains weights that connect the input features with the features in the first hidden layer.

The weight matrix \mathbf{W} undefined is a d \times m undefined matrix, where m undefined is the number of features from the previous layer (or input features) and d undefined is the number of neurons in a layer, or put differnetly d undefined is the number of hidden features produced by the weight matrix for the next layer m^{<l>} = d^{<l+1>} undefined. \mathbf{b} undefined is a d \times 1 undefined vector of biases.

We can calculate the net input \mathbf{Z} undefined by multiplying the matrix of features \mathbf{X} undefined with the transpose of the weights \mathbf{W}^T undefined and adding the weights \mathbf{b} undefined. The result is an n \times d undefined matrix. Finally we apply an activation function (sigmoid in our case) to all net inputs, to get the activation values \mathbf{A} = a(\mathbf{Z}) undefined .

We can calculate the outputs for each of the layers by applying the same procedure over and over again.

For the first layer we get:

\mathbf{Z^{<1>}} = \mathbf{X}\mathbf{W^{<1>T}} + \mathbf{b^{<T>}} \\ undefined
\mathbf{A^{<1>}} = a( \mathbf{Z^{<l>}}) \\ undefined

For all the other layers we get:

\mathbf{Z^{<l>}} = \mathbf{X}\mathbf{A^{<l-1>T}} + \mathbf{b^{<T>}} \\ undefined
\mathbf{A^{<l>}} = a( \mathbf{Z^{<l>}}) \\ undefined

The output of the very last layer L undefined corresponds to the predicted values \hat{\mathbf{y}} undefined. In our case\hat{y} undefined is a scalar value.

The neural network is essentially just a collection of matrix multiplications and nonlinear activation functions. Each matrix multiplication follows by an activation and each activation is followed by a matrix multiplication until the last layer is reached.

Essentially a neural network is a deeply nested function composition, where the output of a function is used as input into the next function.

\mathbf{\hat{y}} = a(\cdots a(a(a(\mathbf{X}\mathbf{W}^{<1>T})\mathbf{W}^{<2>T}) \mathbf{W}^{<3>T}) \cdots \mathbf{W}^{<L>T}) undefined

This nesting makes it harder to determine the gradients of individual weights. Luckily the backpropagation algorithm allows us to calculate the gradients of all the weights and biases in a very efficient manner.

We can demonstrate how this procedure works using the neural network from above.

Let us assume that we receive a batch of inputs containing 3 samples.

\mathbf{ X } = \begin{bmatrix} 0.50 &0.50 \\0.95 &0.50 \\0.75 &0.50 \\ \end{bmatrix} \\ undefined

Whe first layer takes a 3 \times 2 undefined matrix as input and generates a 3 \times 4 undefined matrix. For that we utilize the matrix\mathbf{W} undefined and the vector \mathbf{b} undefined.

\mathbf{ W ^{ <1> }} = \begin{bmatrix} -0.48 &0.12 \\-0.32 &0.01 \\1.61 &0.39 \\-0.44 &-0.60 \\ \end{bmatrix} \\ undefined, \mathbf{b}^{<1>} = \begin{bmatrix} -0.51 \\0.78 \\-1.59 \\0.05 \\ \end{bmatrix} \\ undefined

After multiplying the input features \mathbf{X} undefined with the weight matrix, adding the bias vector and applying the sigmoid \mathbf{A} = a(\mathbf{X}\mathbf{W}^T + \mathbf{b}) undefined we end up with the outputs for the first layer.

\mathbf{A}^{<1>} = \begin{bmatrix} 0.33 &0.65 &0.36 &0.39 \\0.29 &0.62 &0.53 &0.34 \\0.31 &0.63 &0.45 &0.36 \\ \end{bmatrix} \\ undefined

We repeat the process taking the outputs from the first layer as input and using the weights and biases from the second layer.

\mathbf{ W ^{ <2> }} = \begin{bmatrix} 0.77 &0.03 &0.40 &0.80 \\-0.47 &1.10 &-0.74 &0.36 \\ \end{bmatrix} \\ undefined, \mathbf{b}^{<2>} = \begin{bmatrix} 0.34 \\-1.79 \\ \end{bmatrix} \\ undefined

Finally we use the weights and the bias from the third and final layer to attain our predictions.

\mathbf{ W ^{ <3> }} = \begin{bmatrix} 0.16 &-1.14 \\ \end{bmatrix} \\ undefined, \mathbf{b}^{<3>} = \begin{bmatrix} 0.58 \\ \end{bmatrix} \\ undefined

The last activations \mathbf{A}^{<3>} undefined correspond to the probabilities to belong to the category 1. Eventually these outputs are used in the calculation of the loss function, the cross-entropy.

\mathbf{\hat{y}} = \mathbf{A}^{<3>} = \begin{bmatrix} 0.6150 \\0.6219 \\0.6189 \\ \end{bmatrix} \\ undefined
